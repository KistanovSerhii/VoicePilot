# Обновления VoicePilot

- На форме команды появилась кнопка «Удалить»: можно удалить выбранную команду, изменения сразу сохраняются в manifest и список обновляется.
- Добавлена возможность задавать параметры встроенных действий (например, mouseMove, sendKeys) прямо в форме редактирования команды через поле «Параметры действия»; поддерживаются JSON и пары ключ=значение.
## 2025-11-03
- Исправлена загрузка пользовательских модулей при запуске: путь `Modules` теперь вычисляется через `PathOptions.FindSolutionRoot`, поэтому список команд наполняется сразу после старта.
- Добавлен режим голосового ввода: системная команда «Голосовой ввод» включает диктовку, текст вводится в активное окно, выход — по тишине или достижению лимита символов.
- Появилось окно «Настройки диктовки» с параметрами таймаута молчания и ограничения на количество символов.
- На главной форме добавлен индикатор состояний микрофона и кнопка для открытия настроек диктовки.
