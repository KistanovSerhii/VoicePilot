# Руководство пользователя VoicePilot

VoicePilot - это голосовой ассистент для Windows, который позволяет управлять компьютером с помощью голосовых команд без использования клавиатуры и мыши.

## Установка приложения

1. Распакуйте архив "VoicePilot-Installer"
2. Запустите `VoicePilot.App.exe` от имени администратора
3. Рекомендация: Отправить "VoicePilot.App.exe" как ярлык на рабочем столе

## Системные требования

Для корректной работы VoicePilot требуются следующие компоненты:

- Windows 10/11 x64
- .NET 9.0 Runtime
- Микрофон и доступ к звуковой подсистеме
- Офлайн-модель Vosk для распознавания речи (например, `vosk-model-small-ru-0.22`)

### Установка модели Vosk

Этот шаг можно пропустить потому что в архиве модель уже установлена.
1. Скачайте модель с сайта проекта: https://alphacephei.com/vosk/models
2. Распакуйте каталог модели в папку `VoicePilot/models`
3. Убедитесь, что путь к модели соответствует значению `speech:modelPath` в файле `config/appsettings.json` (по умолчанию `models/vosk-model-small-ru-0.22`)

## Как пользоваться приложением

### Основные принципы работы

VoicePilot работает с модульными командами, которые можно создавать и настраивать под свои нужды. Приложение распознает голосовые команды и выполняет соответствующие действия.

### Системные команды

Приложение использует следующие системные команды по умолчанию:
- Активация: "алиса", "слушай"
- Режим тишины: "тишина", "не слушай"
- Выход: "выход", "заверши работу"

### Создание команд

Команды представляют собой JSON-файлы в папке `Modules`. Каждая команда содержит:
- Фразы активации (что нужно сказать для запуска команды)
- Действия (что нужно выполнить)

#### Рекомендации по созданию команд

Для лучшего распознавания голоса при создании команд следуйте этим правилам:

1. Делайте команды запуска и завершения не созвучными, чтобы избежать ошибок распознавания:
   - Плохо: "запусти блокнот" и "заверши блокнот" (начинаются и заканчиваются похожими звуками)
   - Хорошо: "открой блокнот" и "хватит блокнот" (различаются по звучанию)

2. Используйте четкие, непохожие друг на друга фразы для разных команд

3. Учитывайте, что "открой" и "открыть" - это разные команды для системы, поэтому тщательно продумывайте названия команд

В интерфейсе программы вы можете видеть, как названы ваши команды, что поможет вам вспомнить правильное название перед запуском нужной команды.

### Основные возможности

VoicePilot поддерживает широкий спектр действий:

1. **Перемещение курсора мыши**:
   - Команды для движения мыши вверх, вниз, влево, вправо
   - Команда остановки движения мыши
   - Команда клика мышью

2. **Голосовой ввод текста**:
   - Команда "диктовка" для режима голосового ввода текста
   - Текст вводится до тех пор, пока вы говорите, с автоматической остановкой при паузе

3. **Открытие папок в проводнике**:
   - Команды для открытия пользовательских ресурсов (музыка, видео и т.д.)
   - Настройка собственных ресурсов через файл `config/resources.json`

4. **Открытие URL**:
   - Открытие ссылок в браузере по умолчанию
   - Открытие веб-страниц в отдельной форме приложения

5. **Запуск программ**:
   - Запуск любых приложений и программ Windows
   - Выполнение скриптов (.bat, .ps1)

6. **Управление окнами**:
   - Переключение между окнами
   - Сворачивание/разворачивание окон

### Импорт и экспорт команд

Командами можно делиться, выгружая и загружая их в форме создания новой команды:
1. Для экспорта команды сохраните её в виде ZIP-архива
2. Для импорта используйте функцию "Импорт команды" в интерфейсе программы

## Удаление приложения

Для удаления VoicePilot:
1. Запустите `Uninstall-VoicePilot.bat` от имени администратора
2. Дождитесь завершения удаления
